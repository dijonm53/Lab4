Detailed Description

Run step-response output for motor/encoder and plot the results.

This program utilizes a simple GUI with a plot in it. It uses Tkinter to display the GUI. The experiemental results are taken from the Nucleo, which is received by the serial port. This is run multiple times to compare different gain values set by the user

This file uses a template for the GUI, given by Dr. John Ridgely
